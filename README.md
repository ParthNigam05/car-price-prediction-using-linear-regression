# Ford Car Price Prediction using Linear Regression

## Overview
This project focuses on predicting Ford car prices using a Linear Regression model. The workflow includes data cleaning, exploratory data analysis (EDA), feature preprocessing, model training, and performance evaluation.

## Dataset
The dataset contains information about Ford cars, including features such as:
- Model
- Year
- Transmission
- Mileage
- Fuel Type
- Engine Size
- Tax
- MPG
- Price

## Project Workflow
1. Data Loading and Inspection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Encoding and Transformation
5. Train-Test Split
6. Linear Regression Model Training
7. Model Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Results
A Linear Regression model was built to predict car prices based on various vehicle attributes. Model performance was evaluated using standard regression metrics.
